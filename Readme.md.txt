# Nand2Tetris Project

> From Nand Gates to a Fully Functional Computer System

이 저장소는 Nand2Tetris (The Elements of Computing Systems) 코스를 학습하며, 하드웨어 계층부터 소프트웨어 스택까지 직접 구현한 기록을 담고 있습니다.

## Project Highlights

Nand 게이트 하나로 시작하여 현대적인 컴퓨터 아키텍처의 전 과정을 계층별로 구현했습니다.

* **Hardware (HDL):** 기본 논리 게이트부터 ALU, CPU, Memory 및 Computer Chip 설계
* **Software Stack:**
* [직접 구현한 언어 입력]를 이용한 **Assembler** 구현
* Stack 기반의 **VM Translator** 제작
* Jack 객체지향 언어를 위한 **Compiler** 구축


* **OS & App:** 표준 라이브러리(OS) 작성 및 Jack 언어를 이용한 응용 프로그램 개발

## Repository Structure

* `/projects`: 각 챕터별 HDL 설계 및 구현 코드
* `/tools`: (학습용 시뮬레이터 제외, 본인 구현체 위주)
* `Learning_Notes.md`: 하드웨어 및 컴파일러 원리 학습 기록
